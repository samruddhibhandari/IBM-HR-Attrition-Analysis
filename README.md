# IBM HR Analytics — Employee Attrition Analysis

## Overview
End-to-end data science project analyzing employee attrition 
at IBM using EDA, Statistics, and Machine Learning.

**Goal:** Predict which employees are likely to leave and 
identify key factors driving attrition.

## Dataset
- Source: IBM HR Analytics Employee Attrition Dataset (Kaggle)
- 1,470 employees | 35 features | Binary target (Attrition Yes/No)
